requestMembers
Link copied to clipboard

Executes the request on this gateway, returning a flow of MembersChunkEvent responses.

The returned flow is cold, and will execute the request only on subscription. Collection of this flow on a Gateway that is not Gateway.start will result in an IllegalStateException being thrown.

Executing the request on a Gateway with a Shard that does not match the guild id can result in undefined behavior for the returned flow and inconsistencies in the cache.

This function expects RequestGuildMembers.nonce to contain a value, but it is not required. If no nonce was provided one will be generated instead.

open override fun requestMembers(request: RequestGuildMembers): Flow<MembersChunkEvent>
