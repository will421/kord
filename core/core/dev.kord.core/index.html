Unsafe
Link copied to clipboard

A class that exposes the creation of {Entity}Behavior classes.

All functionality in this class assumes correct data is being passed along and omits any requirements or checks. This makes using behaviors created by this class inherently unsafe.

If the user is not sure of the correctness of the data being passed along, it is advised to use Entities generated by Kord or other Entities instead.

@KordUnsafe()
@KordExperimental()

on
Link copied to clipboard

Convenience method that will invoke the consumer on every event T created by Kord.events.

The events are buffered in an unlimitedbuffer and launched in the supplied scope, which is Kord by default. Each event will be launched inside the scope separately and any thrown Throwable will be caught and logged.

The returned Job is a reference to the created coroutine, call Job.cancel to cancel the processing of any further events for this consumer.

inline fun <T : Event> Kord.on(scope: CoroutineScope, noinline consumer: SuspendFunction1<T, Unit>): Job
