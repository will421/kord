DiscordVoiceState

A representation of the Discord Voice State structure. Used to represent a user's voice connection status.

data class DiscordVoiceState(guildId: OptionalSnowflake, channelId: Snowflake?, userId: Snowflake, member: Optional<DiscordGuildMember>, sessionId: String, deaf: Boolean, mute: Boolean, selfDeaf: Boolean, selfMute: Boolean, selfVideo: Boolean, selfStream: OptionalBoolean, suppress: Boolean)
