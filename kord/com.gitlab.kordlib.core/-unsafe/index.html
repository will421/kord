Unsafe

A class that exposes the creation of {Entity}Behavior classes.All functionality in this class assumes correct data is being passed along and omits any requirements or checks. This makes using behaviors created by this class inherently unsafe.If the user is not sure of the correctness of the data being passed along, it is advised to use Entities generated by Kord or other Entities instead.
@KordUnsafe()
@KordExperimental()
class Unsafe(kord: Kord)
